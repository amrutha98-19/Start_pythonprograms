 Topic:Dictionary
 ***********
   A dictionary is a collection of data stored in key-value pairs.

    - A key is used to identify the data.
    - A value is the data associated with the key.


 Features
 **********
  - Stores data as key-value pairs
  
  - Mutable
  - Keys are unique [Does not allow duplicate keys]
  - Values can be duplicated [Multiple keys can have the same value.]
  - Created using curly braces {}
  - Can store different data types [Values can be strings, integers, lists, tuples, etc.]
  - Dynamic size [Items can be added or removed at any time.]
  - Fast access using keys[ Values can be accessed directly using their keys.]
