As usual, continuing the Python learning journey with Functions in Python.
    * A function is a reusable block of code used to perform a specific task.
    * Functions help reduce code repetition.
    * Functions make programs easier to read and maintain.
    * Functions improve code reusability and organization

Think of a function like making a cup of tea:
    * You follow the same steps every time to make tea.
    * Boil water.
    * Add tea powder.
    * Add sugar and milk.
    * Serve the tea.
    * Whenever you want tea, you repeat the same process to get the same result.

Similarly, in Python, a function allows you to write a set of instructions once and use them whenever needed.

Why use functions?
***********************

    * Code reusability
    * Better readabillity
    * Easy debugging
    * Code optimization
