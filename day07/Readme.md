## Day 7 -> Modules
**********************

A Python module is a .py file.

It contains Python code such as:

- Functions
- Variables
- Classes
- Statements

Modules help in **reusing code**.

The same module can be used in **multiple Python programs**.

Modules make the code **organized** and **easy to maintain**.

## Topics Covered
*********************
- Create a module
- Import the entire module
- Import specific functions or variables
- Import all items from a module
- Import a module with an alias
- Import a specific function with an alias
- Built-in modules in Python
- math module
- random module
- time module
- datetime module

## Goal
*************

- Understand module usage in Python.
- Learn different ways to import modules.
- Practice using built-in and user-defined modules.
